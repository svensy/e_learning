Welcome to EElearning!

First you need to install ruby 2.2.4p230 [i386-mingw32] for windows 7 32bit.

To clone:
1) git clone https://github.com/svensy/EElearning.git
2) bundle install
3) rake db:create
4) rake db:migrate
5) rails s
